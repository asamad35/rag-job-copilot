// turbo-all

All terminal commands in this project are safe to auto-run. This includes:

1. Install dependencies (`pnpm install`, `npm install`, etc.)
2. Run dev server (`pnpm dev`, `npm run dev`)
3. Run builds (`pnpm build`, `npm run build`)
4. Run type checks (`pnpm typecheck`)
5. Run tests and evaluation scripts
6. File operations (create, move, delete)
7. Git operations
